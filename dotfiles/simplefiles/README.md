SimpleFiles has a real config file.

Install target:

    ~/.config/simplefiles/config

Supported keys in current SimpleFiles include:

    OPEN_MODE
    EDITOR_COMMAND
    TEXT_EXTENSIONS
    PREVIEW
    PREVIEW_LINES
    TRASH_DIR
    CONFIRM_DELETE
    OPEN_.ext

SimpleFiles starts in the current working directory. Pass a directory path on
the command line when it should start elsewhere; `START_DIR` is no longer a
configuration key.
